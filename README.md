## ✨ 项目简介

本项目致力于收集和整理大连海事大学各学院的课程资料，包括但不限于课程笔记、复习资料、历年试题、实验报告等。旨在为DMU学子提供全面的学习支持，促进学术交流与知识共享。

### 🎯 项目愿景
受以下优秀项目启发，我们希望建立一个属于DMU学子的课程资源共享平台：
- [清华大学计算机系课程攻略](https://github.com/PKUanonym/REKCARC-TSC-UHT)
- [浙江大学课程攻略共享计划](https://github.com/QSCTech/zju-icicles)
- [上海交通大学课程攻略](https://github.com/kxxwz/SJTU-Courses)

## 📖 使用指南

### 🚨 重要声明
1. **版权说明**：本仓库所有资料**仅供个人学习交流**使用，请在下载后24小时内删除
2. **非商业用途**：禁止将本仓库内容用于任何商业用途
3. **内容时效性**：部分资料可能存在时效性问题，请结合最新课程要求使用
4. **责任声明**：使用者需自行判断资料的适用性，作者不承担任何责任

### 📝 使用建议
- 建议优先查看最新版本的资料
- 欢迎补充和更新过时的内容
- 如发现错误，请及时提出Issue

## 🤝 贡献指南

我们热烈欢迎所有DMU学子的贡献！无论是补充资料、修正错误还是提出建议，都是对项目的宝贵支持。

### 📌 贡献方式

#### 1. 提交 Pull Request
请参考：[Pull Request 攻略](https://njuptfreeexams.github.io/UploadGuide.html)
- 建议上传PDF格式文件
- 单个文件请勿超过100M
- 确保内容清晰可读

#### 2. 提交 Issue
请参考：[Issue 指南](https://njuptfreeexams.github.io/IssueGuide.html)
- 报告资料错误或缺失
- 提出功能建议
- 反馈使用问题
